🚀 SignalR Practice Application

This application was built with the purpose of practicing the use of the SignalR library together with Angular and .NET Core.

📌 Implemented examples:

  1. Real-Time Restaurant: it has two main screens:
    • 👤 Customers – interface for customers to place their orders.
    • 👨‍🍳 Kitchen – kitchen panel where all customer orders are displayed and their status can be updated in real time.
  2. Real-Time Group Chat: it has one main screen:
    • 💬 Chat – interface where you can enter a group chat and send messages.💬

Project created as a learning exercise with SignalR.
